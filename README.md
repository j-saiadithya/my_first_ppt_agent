# my_first_ppt_agent
It is an agent which is designed to show how the work flow (the thinking process of the agents ) and to get better understanding of how agent ai and mcp architecture




##  Overview

This project builds an AI Agent that automatically creates PowerPoint presentations from a single prompt using MCP (Model Context Protocol).

##  Objective

Convert user input like:
"Create a 5-slide presentation on AI"

Into a fully functional `.pptx` file.

##  Features

* Agentic planning before execution
* MCP-based tool usage
* Automatic slide generation
* Error handling for vague inputs

##  Architecture

User Prompt → Agent → MCP Tools → PPT File

##  Tech Stack

* Python
* MCP
* LangChain / LangGraph
* python-pptx

##  Folder Structure

agent/ → AI Agent
mcp_servers/ → MCP tools
output/ → Generated PPT
